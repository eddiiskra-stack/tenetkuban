# **App Name**: Tenet Configurator

## Core Features:

- HTML Cleanup and Rebranding: Clean up the HTML structure from the provided code, replacing all mentions of 'Chery' with 'Tenet'.
- SEO Meta Tags: Implement SEO meta-tags (title, description, keywords) based on the provided semantic core for improved search engine visibility.
- Critical CSS: Optimize CSS delivery by inlining critical CSS or including it in a way that does not block rendering, improving initial load time.
- Car Cost Calculator: Implement a calculator that allows users to select a Tenet model (4, 7, 8, 9) and trim level (e.g., Base, Comfort, Flagship) to calculate the total price.
- Lead Form Submission: A 'Get Offer' button triggers a modal form. Once submitted, user data (Name, Phone, Model Choice) is securely sent to Firebase Firestore.
- Firestore Integration: Integrate Firebase Firestore to store user-submitted form data, ensuring reliable data capture and easy access for the sales team.
- CI/CD Pipeline: Automate deployments with a CI/CD pipeline using GitHub Actions to build and deploy changes to Yandex Cloud, ensuring rapid and reliable updates.

## Style Guidelines:

- Primary color: Deep red (#A61E21) evoking a sense of boldness and reliability, complementing the automotive industry's dynamic nature.
- Background color: Light gray (#F2F2F2) providing a neutral and clean backdrop that emphasizes the content.
- Accent color: Dark red (#7A1416) used sparingly for key interactive elements such as call-to-action buttons and links to guide user attention.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, to convey a tech-forward feel.
- Body font: 'Inter' (sans-serif) for body text where longer text is used; complements the headline font while promoting readability.
- Modern, geometric icons to represent car models and features, providing a clean and intuitive user experience.
- Clean, spacious layout with a focus on readability and user-friendly navigation. Responsive design to adapt seamlessly across devices.
